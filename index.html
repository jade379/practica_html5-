Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web.

Comenzar

¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado de hipertexto (HTML), el estándar que se utiliza para crear y estructurar el contenido de las páginas web. No es un lenguaje de programación, sino de marcado, lo que significa que se utiliza para definir la estructura y el significado del contenido mediante etiquetas. Sirve para organizar textos, imágenes, videos, enlaces y otros elementos, y es la base sobre la que se construyen todos los sitios web. Es importante porque permite crear sitios más modernos, accesibles y compatibles con diferentes dispositivos, además de integrar de forma nativa multimedia y nuevas funcionalidades sin necesidad de plugins externos.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

!DOCTYPE html: Indica al navegador qué versión de HTML se está usando, en este caso HTML5.
html: Es la etiqueta raíz que contiene todo el código de la página.
head: Aquí se coloca información que no se ve directamente en la página, como el título, la codificación de caracteres y enlaces a estilos o scripts.
body: Es el área visible de la página, donde se coloca todo el contenido que verá el usuario: textos, imágenes, videos, etc.

Etiquetas principales

h1 - h6

Son etiquetas para definir títulos y subtítulos, siendo h1 el más importante y grande, y h6 el menor.

p

Se usa para escribir párrafos de texto normal.

img

Sirve para insertar imágenes en la página, usando el atributo src para indicar la ruta de la imagen.

a

Permite crear enlaces o hipervínculos hacia otras páginas o secciones, usando el atributo href.

button

Crea botones interactivos que el usuario puede presionar.

ul / ol / li

Se usan para crear listas: ul para listas sin orden (con viñetas), ol para listas ordenadas (con números) y li para cada elemento de la lista.

HTML5 semántico

header

Representa el encabezado de la página o de una sección, generalmente contiene logotipos, títulos y menús.

nav

Se utiliza para agrupar los enlaces de navegación del sitio.

main

Indica el contenido principal y más importante de la página.

section

Define una sección temática dentro del documento, agrupando contenido relacionado.

article

Representa un contenido independiente y completo, como un artículo, una noticia o un post.

footer

Es el pie de página, donde suele ir información de contacto, derechos de autor, etc.

Formularios en HTML5

Los formularios son elementos que permiten al usuario ingresar datos y enviarlos al servidor para ser procesados. Algunas etiquetas importantes son:
form: Contiene todo el formulario.
label: Describe qué dato se debe ingresar en un campo.
input: Campo para introducir datos, puede ser de diferentes tipos como texto, email, contraseña, etc.
textarea: Área para escribir textos más largos.
select: Menú desplegable para elegir una opción.
button: Botón para enviar o resetear el formulario.

Multimedia en HTML5

HTML5 permite integrar contenido multimedia de forma sencilla y nativa, sin necesidad de programas adicionales.

Imágenes

Se usa la etiqueta img, con atributos como src (ruta de la imagen) y alt (texto descriptivo).

Audio

La etiqueta audio permite reproducir archivos de sonido directamente en el navegador.

Video

Con la etiqueta video se pueden reproducir videos sin plugins externos.

Accesibilidad

El atributo alt en las imágenes es muy importante porque describe la imagen para personas que no pueden verla, mejorando la accesibilidad del sitio.

Conclusión

En conclusión, HTML5 es fundamental en el desarrollo web actual porque ofrece una estructura clara, semántica y moderna para crear sitios web. Facilita la organización del contenido, mejora la accesibilidad y permite integrar multimedia y formularios de manera eficiente. Conocer y usar correctamente sus etiquetas y características es esencial para cualquier persona que quiera dedicarse al desarrollo web, ya que es la base sobre la que se construyen todas las aplicaciones y páginas que usamos hoy en día.
